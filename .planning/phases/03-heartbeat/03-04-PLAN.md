# Plan 3.4 — Heartbeat API

**Goal:** REST API endpoints for heartbeat operations.

**Requirements:** BEAT-07, API-03

**Assignment:** @backend-architect

## Implementation

### 3.4.1 Agent Registration API

Create `ethos_os/api/agents.py`:
- POST /agents — Register new agent
- GET /agents/{agent_id} — Get agent details
- GET /agents — List agents

### 3.4.2 Heartbeat Recording API

Add to `ethos_os/api/agents.py`:
- POST /agents/{agent_id}/heartbeat — Record heartbeat
- GET /agents/{agent_id}/heartbeats — Get heartbeat timeline

### 3.4.3 Agent Status API

- GET /agents/{agent_id}/status — Get current status
