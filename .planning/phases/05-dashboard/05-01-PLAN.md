# Plan 5.1 — Initiative Tree View

**Purpose:** Read-only nested tree display of the initiative hierarchy.

**Scope:**
- GET /tree endpoint already exists in hierarchy API
- Add frontend component to display nested tree
- Support expand/collapse at each level
- Color coding by status

**Implementation:**
- Add HTML template with JavaScript tree rendering
- Fetch from /hierarchy/tree endpoint
- Render as collapsible nested list
- Apply status-based colors (approved=green, pending=yellow, rejected=red)

**Dependencies:** API-01 (existing)
